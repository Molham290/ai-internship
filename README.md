# AI Engineering 7-Week Summer Training Bootcamp

This repository serves as a comprehensive portfolio documenting my journey through the 7-week AI Engineering Summer Training program. The course transitions from basic prompt engineering to building fully autonomous AI agents. 

## 🛠️ Tools & Technologies
Throughout the training, we utilize a powerful stack of tools:
* **Python:** For scripting and logic.
* **Git:** For version control.
* **Ollama:** For local model inference.
* **Claude:** For advanced skill building.
* **n8n:** For workflow automation.

---

## 📅 Curriculum Overview

### Week 1
* Learned AI foundations and prompt engineering.
* Crafted effective zero-shot and few-shot prompts.

### Week 2
* Explored Vibecoding and AI pair programming.
* Developed a local Security+ Quiz Agent using Llama 3 and Gradio.

### Week 3
* Focused on context engineering.
* Formatted Markdown files for LLMs.

### Week 4
* Shifted to using and building Claude Skills.
* Created custom `SKILL.md` files.

### Week 5
* Studied Agentic AI Fundamentals.
* Explored how agents use tools, memory, and reasoning loops like ReAct.

### Week 6
* Dived into Automation and Orchestration.
* Used n8n to connect various APIs and create intelligent workflows.

### Week 7
* Culminated in a Capstone Project.
* Built and deployed a complete Agentic AI application integrating all previous concepts.

## 🤖 Chatbots vs. AI Agents
To understand the shift in this bootcamp, here is a comparison between standard Chatbots and AI Agents:

| Feature | Chatbots | AI Agents |
| :--- | :--- | :--- |
| **Behavior** | Reactive | Proactive |
| **Memory** | Lacks long-term memory | Maintains memory |
| **Actions** | Cannot execute external actions | Can use external tools via function calling |
